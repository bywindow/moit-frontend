### 📌 Description
"모잇"은 대학생을 위한 단건배달 플랫폼이다.<br>
해당 Repository는 "모잇" 모바일 에플리케이션과 연동하여 음식점에서 주문을 받을 수 있는 WEB application이다.
<br>

## Installation
---
```bash
$ npm intall
```

## Run
---
```bash
# in project folder
$ npm start
```
## Release
---
```bash
$ npm install -g serve
$ npx serve -s build
```
